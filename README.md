OHAI – 目覚ましアプリ紹介 LP
■ 公開URL

https://brother0502tkm-beep.github.io/portfolio01/

■ リポジトリ

https://github.com/brother0502tkm-beep/portfolio01

■ 概要

目覚ましアプリ 「OHAI」 の紹介を目的としたコーポレートサイト風 LP です。
オンラインスクールの課題として、デザインカンプの再現性 と
レスポンシブ対応の習得 を目的に制作しました。

イラストを多用したデザインのため、視認性・余白・動きの心地よさ に重点を置いてコーディングしています。

■ 制作の目的・背景

Figma カンプを元にした デザイン再現力向上

コーポレート系 LP の構造理解

JavaScript を使用した簡易アニメーション・ハンバーガーメニューの実装練習

スマホファーストを意識したレスポンシブ対応

■ 使用技術

HTML

CSS / Sass

JavaScript

■ こだわったポイント

ハンバーガーメニュー の開閉アニメーション

画像・イラストを邪魔しない 自然なフェードインアニメーション

Figma カンプのレイアウトを崩さないよう、
余白・行間・サイズ比率の再現を徹底

スマホ版の UI を考慮した レスポンシブ設計

Sass を使った 変数管理・ネスト構造・mixin の活用

■ 制作期間

約 2 週間

（デザイン分析 → コーディング → JS 実装 → レスポンシブ対応）

■ 担当範囲

HTML / CSS / Sass コーディング全般

JavaScript（ハンバーガーメニュー、フェード演出など）

スマホ・タブレット向けのレスポンシブ対応

Figma カンプの構造理解・再現
