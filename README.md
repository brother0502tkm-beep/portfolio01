# OHAI – 目覚ましアプリ紹介 LP

## ■ 公開URL
https://brother0502tkm-beep.github.io/portfolio01/

## ■ リポジトリ
https://github.com/brother0502tkm-beep/portfolio01

---

## ■ 概要
目覚ましアプリ **「OHAI」** の紹介を目的としたコーポレートサイト風ランディングページ（LP）です。  
オンラインスクールの課題として、**Figma デザインカンプの再現性** と **レスポンシブ対応の習得** を目的に制作しました。  
イラストを多用したデザインのため、見やすさ・余白の取り方・動きなどに重点を置いてコーディングしています。

---

## ■ 制作の目的・背景
- Figma カンプを元にしたデザイン再現力の向上  
- コーポレート系 LP の構造理解  
- JavaScript を用いた簡易アニメーション・ハンバーガーメニュー実装の練習  
- スマホ版を意識したレスポンシブ対応

---

## ■ 使用技術
- HTML  
- CSS / Sass（SCSS）  
- JavaScript

---

## ■ こだわったポイント
- ハンバーガーメニューの開閉アニメーション  
- 画像・イラストを邪魔しない自然なフェードインアニメーション  
- Figma カンプの余白・行間・サイズ比率の忠実な再現  
- スマホ版の UI を意識したレスポンシブ設計  
- Sass による変数管理・ネスト・mixin の活用で保守性を向上

---

## ■ 制作期間
約 2 週間  
（デザイン分析 → コーディング → JavaScript 実装 → レスポンシブ調整）

---

## ■ 担当範囲
- HTML / CSS / Sass コーディング全般  
- JavaScript（ハンバーガーメニュー、フェード演出 等）  
- スマホ・タブレット向けレスポンシブ対応  
- Figma カンプの構造理解・再現

---

## ■ 備考（確認事項）
- 表示確認は PC とスマホで行っています。  
- スクリーンショットは PC ビュー優先で掲載しています（スマホ版は必要な場合に追加可能）。
